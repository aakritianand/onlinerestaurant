Project Title: Tu Phuong Restaurant

Description:
Tu Phuong Restaurant is a web application for developing online restaurant on web server. It brings interactions between restaurant and customers to a online platform, makes them wider and faster than any offline interactions.
Public information about Tu Phuong Restaurant and menu are updated easily and as early as possible on the online application. All guests can check the information, booking and ordering directly.

Actors:
Each level of users has different authorities to set up and access on the online resstaurant.
- Main admin
- Sub admin
- Non-registered guest
- Registered guest

Built with:
- React JS
- NodeJS
- Miscrosoft SQL Server
